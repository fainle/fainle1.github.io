    第一周 Introduction

    第一课 Welcome

    每天都在用不同的机器学习算法生活，

    • 包括谷歌搜索排名(nlp)
    • facebook推荐朋友的照片 (图识别，和推荐算法)
    • 垃圾电子邮件过滤 (贝叶斯概率)
    • 最终实现和人类智商一样的强人工(愿景 学习算法？强化学习？)

    注意事项:

    • 仅学习数学和算法还不够。
    • 需要用编程解决实际问题

    第二课 What is machine learning

    定义：

    • 一个程序被认为能从经验E中学习，解决任务T，达到性能度量P, 当且仅当，有了经验E后，经过p判断，在处理T时性能有所提升。

    第三课 监督学习 (Supervised Learning)

    给出一个算法，需要部分数据已有正确答案。比如给定房价数据集，即这房子实际卖出的价格，算法的结果就是，算出更多的价格。如果输出的值是一个连续值（回归）有一个拟合的线来预测值，输出的值是一个离散值（分类）有一根分类的线来区分开值。

    可以有无限个特征。（可以用一些算法做处理，比如svm）

    分类可能有多个输出项。

    回归的例子：预测公司的销量。

    分类的例子：预测公司的无性别账号可能是男还是女？

    第四课 非监督学习 Unsupervised Learning

    给定一堆数据，但并没有正确的答案，需要算法自己去寻找它可能的分类。（就是监督学习）

    举例:

    • 大量的新闻分类。
    • 寻找社交网络的朋友分组
    • 寻找客户的细分市场。
    • 混乱的环境里面，提取出每个人的音频。
