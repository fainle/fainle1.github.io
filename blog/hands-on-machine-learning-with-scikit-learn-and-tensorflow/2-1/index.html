  • 端到端机器学习 End-to-End Machine Learning Project

    第一步：获取数据 。

    第二步：观察数据和可视化数据。

    第三步：为机器学习算法准备数据。（清洗）

    第四步：选择一个最合适的模型，并跑起来。

    第五步：调优模型。

    第六步：提出你的解决方案。

    第七步：部署，监控和维护系统。

    注意事项：

    “is it supervised, unsupervised, or Reinforcement Learning?”

    弄清除问题属于 监督学习 无监督学习 还是强化学习

    “Is it a classification task, a regression task, or something else?”

    弄清楚属于 分类问题 还是 回归问题 或者其他

    “Should you use batch learning or online learning techniques?”

    弄清楚 离线学习或者在线学习
